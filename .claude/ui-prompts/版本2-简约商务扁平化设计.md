# 版本2：简约商务 - 扁平化设计

## 项目背景
为浙江省舟山市普陀区沈西派出所设计警务智管助手大屏可视化系统，技术栈为Vue 3 + DataV Vue3 + ECharts + 天地图API。

## 设计要求

### 整体风格
简约商务风格，扁平化设计，强调信息层级和可读性，适合长时间观看

### 颜色方案
- **主色**：#2563eb（现代蓝）、#1e40af（深蓝）
- **辅助色**：#10b981（成功绿）、#f59e0b（警告橙）、#ef4444（危险红）
- **中性色**：#f8fafc（浅灰背景）、#64748b（文字灰）
- **渐变**：线性渐变，从浅到深

### 视觉元素
- 使用DataV的 `dv-border-box-6`、`dv-border-box-7`（简洁边框）
- 大圆角卡片（border-radius: 16px）
- 扁平化图标（Remix Icon）
- 阴影层次：`box-shadow: 0 4px 6px rgba(0,0,0,0.1)`

## 设计规范

### 间距系统
- 超小：4px
- 小：8px
- 中：16px
- 大：24px
- 超大：32px

### 圆角系统
- 小：8px（按钮、输入框）
- 中：12px（小卡片）
- 大：16px（大卡片）
- 超大：20px（胶囊形）

### 阴影系统
- 浅：`0 2px 4px rgba(0,0,0,0.06)`
- 中：`0 4px 12px rgba(0,0,0,0.08)`
- 深：`0 8px 24px rgba(0,0,0,0.12)`

### 字体系统
- 超大标题：48px / 700
- 大标题：28px / 700
- 中标题：20px / 600
- 正文：16px / 400
- 小字：14px / 400
- 超小字：12px / 400

## 开发注意事项

1. **保持简洁**
   - 避免过度装饰
   - 突出信息层级
   - 留白充足

2. **颜色使用**
   - 主色用于重要操作
   - 辅助色用于状态提示
   - 中性色用于背景和文字

3. **可访问性**
   - 对比度符合WCAG标准
   - 支持键盘导航
   - 提供文字说明
