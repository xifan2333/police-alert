# 沈西所警情态势演示系统

## UI基调

- 色调：警务蓝
- 审美偏好：科技感/严肃/正式

## 功能

### 主界面

- 主标题 沈西所警情态势演示系统
- 右侧纵向排布四个按钮，导向二级页面
    - 警员定位
    - 警情态势
    - 未办结案件
    - 未调解纠纷

### 警员定位

- 跳转至第三方页面即可
- url：https://www.azliot.com/home/siteMap

### 警情态势

- 中间为地图组件，使用天地图 api，需社区显示辖区内的个社区的案情和纠纷分布情况
- 左侧为案情列表
- 右侧为纠纷列表

### 未办结案件

- 列表，滚动模式

### 未调解纠纷

- 列表，滚动模式

## 其他说明

- 当前项目仅仅为演示 demo 供甲方确定 UI 方案，应该先保证界面美观，交互正常，数据准确
